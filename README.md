# sales-analytics-dashboard
📌 Problem Statement

This project analyzes sales data to understand revenue performance, customer behavior, product trends, and operational efficiency using an end-to-end data pipeline.
🛠️ Tech Stack
Python (Pandas, NumPy)
SQL (Star Schema Modeling)
Power BI (Dashboard & Visualization)
Git & GitHub (Version Control)
🗂️ Data Model (Star Schema)
fact_sales (Transaction data)
dim_customer (Customer details)
dim_product (Product details)
dim_date (Date dimension)
📊 Key KPIs
💰 Business KPIs
Total Revenue
Average Order Value (AOV)
Revenue Growth Rate
📦 Sales KPIs
Total Orders
Top Products
Orders per Customer
🔁 Operational KPIs
Return Rate %
Average Delivery Time
Order Completion Rate
💡 Customer KPIs
📈 Dashboard Features
Executive KPI Overview
Product Performance Analysis
Customer Segmentation Analysis
Operational Insights (Delivery & Returns)
🔍 Key Insights
Top products contribute major revenue share
Certain customer segments drive majority of revenue
Return rate impacts overall profitability
Delivery time affects customer satisfaction
🚀 Conclusion

This project demonstrates an end-to-end data analytics workflow from raw data cleaning to business insights using Python, SQL, and Power BI.
